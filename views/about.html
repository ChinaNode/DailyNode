Intro

DailyNode is a aggregator of Node.js things: News, Blog, Video, Tutorial, Module, Framework, Tool. The content mainly is crawled from internet and RSS. Now we have over 2K items. We want it become the biggest, completest, timely Node.js hub.

Tech

DailyNode is build on Koa.js and run on v0.11.13. Content was crawled with feedParser and cheerio. Here is the detail tech stack:

  • Node v0.11.13
  • Koa
  • MongoDB
  • Mongoose
  • feedParser
  • cheerio

Intent

We want build a place where you can know what's happening in Node.js land. And the newest tech use about Node.js. We build it because there are a lot Noder like me cared about Node.js. We hope Node.js can grow well, more developer use and love it.
